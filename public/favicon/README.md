# Favicon Folder

Place your favicon files in this folder:

## Recommended Files:
- `favicon.ico` - Main favicon (16x16, 32x32, 48x48)
- `favicon.svg` - Scalable vector favicon (modern browsers)
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `apple-touch-icon.png` - 180x180 PNG for iOS devices

## Usage:
1. Add your favicon files to this folder
2. The favicons will be automatically referenced in `index.html`
3. You can also import them in your components using `/src/assets/favicon/index.js`

## File Naming:
Make sure your files match the names referenced in:
- `index.html` (for direct HTML references)
- `src/assets/favicon/index.js` (for component imports)
